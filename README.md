# todo-isomorphic
A Todo-list application to practice front and back end. Technologies used:

* React and Flux
* Express
* MongoDB


## Usage
Clone it:
```
$ git clone https://github.com/lifeBalance/todo-isomorphic .
```

Install it:
```
$ cd your-react-project && npm install
```

And start the thing:
```
$ npm start
```
